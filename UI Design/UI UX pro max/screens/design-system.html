13 — Motion Guide
Fast — 100ms
--t-fast: 0.1s ease
Hover states, colour transitions, icon colour changes. Imperceptible but prevents flicker.
Normal — 200ms
--t-normal: 0.2s ease
Sidebar collapse, dropdown open, slide panels. Feels responsive.
Slow — 300ms
--t-slow: 0.3s ease
Modal entry, page transitions, SLA ring fill. Deliberate, communicates state change.
Standard Entry Animation
Elements entering the viewport use fade-in + translateY(4px → 0) at 250ms with ease easing. Cards animate in sequence with 40ms stagger when multiple appear simultaneously.
1@keyframes fadeIn {
2  from { opacity: 0; transform: translateY(4px); }
3  to { opacity: 1; transform: translateY(0); }
4}
5/* 250ms, animation-fill-mode: both */
